Fleet Dashboard
Aircraft Type Page Builder
Generates a configs/ JSON file for any aircraft type
How It Works
1
Fill in the aircraft type details, inspection intervals, and file names below.
2
Click Generate Config and download the JSON file.
3
Save it as configs/<your-aircraft-type>.json in the repo.
4
Upload your Due-List CSV and Component Change CSV to data/.
5
Run the generator: python scripts/fleet_dashboard_generator.py --config configs/your-type.json — or trigger it via GitHub Actions.
Inspection tracking modes
HOURS Track by flight hours only (most common).
DAYS Track by calendar days only (e.g. annual, 6-year).
BOTH Tracked by both — whichever comes first in CAMP.
